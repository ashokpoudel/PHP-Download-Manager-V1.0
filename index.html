PHP-Download-Manager-V1.0 by ashokpoudel

PHP Download/Upload Manager Fully implemented in Codeigniter , Datamapper ORM, TankAuth , Bootstrap CSS Framework, MySQL and blueimp jQuery File Upload

Being entirely new ,Sorry if any mistakes and immature steps ... I watched all over the net for a suitable upload and download manager having user groups features and database handling. But i ended up with incomplete or singular files and projects with flat files . So , i came up with all this available features , joined them all and added a functionality. hope somebody will take an advantage of it someday. Projects certainly has lacking functionality but will certainly be added with everybody's help.
